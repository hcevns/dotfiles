# Dotfiles

## Setup

```
# macOS
xcode-select --install
git clone https://github.com/hcevns/dotfiles.git ~/.dotfiles
sh ~/.dotfiles/install.sh
sh ~/.dotfiles/configure-macos.sh

# Arch Linux
pacman -S git
git clone https://github.com/hcevns/dotfiles.git ~/.dotfiles
sh ~/.dotfiles/install.sh
sh ~/.dotfiles/configure-arch.sh
```
